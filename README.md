# 💻 CodSoft Internship – Python Development

Completed a CodSoft Internship in Python Development, working on multiple real-world projects that enhanced my programming, problem-solving, and project management skills. Gained hands-on experience with APIs, file handling, and OOP while developing efficient and structured Python applications.

This repository contains the projects I developed during my **CodSoft Internship** in **Python Development**.  
Each task helped me enhance my skills in logic building, Python programming, and project structuring.

## 🚀 Key Learnings
- Strengthened problem-solving and coding logic  
- Implemented Python concepts like OOP and file handling  
- Gained practical experience in building user-friendly applications

- 🧮 **Calculator** – A simple GUI calculator for basic arithmetic operations  
- 📞 **Contact Book** – Manage, search, and update contact details efficiently  
- 🔐 **Password Generator** – Generate strong, secure passwords dynamically
